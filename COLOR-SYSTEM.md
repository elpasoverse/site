# El Paso Verse - Official Color System

## 9 Core Colors

### Primary Colors
```
Dark Brown:     #3A2820  // Main text, dark elements
Primary Gold:   #C9A961  // Accents, links, borders
Bright Gold:    #D4A745  // Highlights, hover states
```

### Secondary Browns
```
Medium Brown:   #8B4513  // Secondary text, subheadings
Light Brown:    #8B7355  // Tertiary text, muted elements
```

### Backgrounds
```
Light Cream:    #F5E6D3  // Main background
Darker Cream:   #EDD9C0  // Section backgrounds, cards
White:          #FFFFFF  // Input fields, cards
```

### Special
```
Very Dark:      #1a1a1a  // Film section, footer backgrounds
```

---

## Usage Guidelines

### Text Colors
- **Primary text:** `#3A2820` (dark brown)
- **Secondary text:** `#8B4513` (medium brown)
- **Muted text:** `#8B7355` (light brown)
- **Light text on dark:** `#FFFFFF` (white)

### Accent Colors
- **Primary accent:** `#C9A961` (gold)
- **Hover/highlight:** `#D4A745` (bright gold)
- **Borders:** `#C9A961` (gold)

### Backgrounds
- **Main sections:** `#F5E6D3` (light cream)
- **Alternate sections:** `#EDD9C0` (darker cream)
- **Cards/boxes:** `#FFFFFF` or `#EDD9C0`
- **Dark sections:** `#1a1a1a` (very dark)

---

## Fonts

### Primary Font
**Playfair Display**
- All headings (h1, h2, h3)
- Emphasized text
- Decorative elements

### Body Font
**Georgia**
- Body text
- Paragraphs
- Secondary content

### Fallbacks
```css
font-family: 'Playfair Display', Georgia, serif;  /* Headings */
font-family: Georgia, 'Times New Roman', serif;   /* Body */
```
